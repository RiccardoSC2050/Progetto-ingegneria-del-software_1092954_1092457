# Gestione Dipendenti Azienda – Applicazione

Questa sezione spiega come scaricare e utilizzare esclusivamente l’applicazione, senza dover scaricare l’intero progetto GitHub.

## Scaricare solo la cartella dell’applicazione

Per evitare di scaricare tutto il repository, è possibile utilizzare il servizio:

https://download-directory.github.io/

Questo strumento consente di scaricare una singola cartella di un progetto GitHub in formato ZIP.

### Procedura

1. Accedi al repository del progetto su GitHub.
2. Apri la cartella:

```
applicazione_scaricabile
```

3. Copia l’URL completo della cartella dalla barra del browser.
4. Vai su:

```
https://download-directory.github.io/
```

5. Incolla l’URL copiato e premi Invio.
6. Verrà scaricato un file ZIP contenente solo la cartella selezionata.
7. Estrai il contenuto dello ZIP.

A questo punto avrai a disposizione esclusivamente i file necessari per l’avvio dell’applicazione.

---

## Avvio dell’applicazione

Dopo l’estrazione troverai due possibilità:

- `AppProgetto-1.1` → installer Windows
- `AppProgetto_0` → cartella dell’app pronta all’esecuzione

### Metodo consigliato – Installazione

1. Apri il file:

```
AppProgetto-1.1
```

2. L'istallazione è immediata senza configurazione.

Al termine dell’installazione, l’applicazione sarà disponibile nel percorso:

```
C:\Programmi\AppProgetto
```

All’interno della cartella si trova l’eseguibile dell’app.

---

### Metodo alternativo – Avvio diretto

1. Apri la cartella:

```
AppProgetto_0
```

2. Avvia l’eseguibile presente al suo interno.

---

Non è richiesta alcuna configurazione aggiuntiva. L’applicazione è pronta all’uso.
